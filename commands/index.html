commands for twitch.tv/twitchcontrolsmypc2

The essentials:

left/right/up/down (moves pointer)

click (left clicks mouse)

gclick (left clicks mouse)


type {text} (types text example: type super sonic racing)

enter, backspace



Info

It could take some time before your command executes because of stream delay.

Use a pc for your own sake, mobile delay is horrible. (especially ios)

If twitch prohibits you from saying the same command over and over again. Differentiate your messages by using capitals. The bot will still pick this up.


!modalert {info} (alerts the mods in discord if something weird is happening with stream. Info is optional, but if you type anything after !modalert, it will be relayed to the mods.)


Keyboard:

type {text} (types text example: type among us imposter)

gtype {text} (types text with better support for games example: gtype wwssdwdasewdw)

ptype {text} (types text with better support for other languages or emojis example: ptype 超級音速賽車)

copy (copies text)

paste (pastes text)


enter (presses enter)

tab (presses tab)

space (presses space)

escape (presses escape key)

backspace (presses backspace)

backspace {n} (presses backspace n times)

win (presses windows key)

page up/down (presses scroll keys)


arrow down/up/right/left (presses arrow keys)

w/a/s/d for {seconds} (holds w,a,s or d for up to 10 seconds example: w for 4)

arrow down/up/right/left for {seconds} (holds arrow keys for up to 10 seconds example: arrow down for 4)


alt tab (presses alt tab)

alt {n} tab (presses alt and then presses tab n times, useful for switching to any app)

alt browser tab (switches a chrome/browser tab)

select all (ctrl A)

new tab (presses control t)

refresh (refreshes browser tab)

control k / tayne (presses control k)

save (presses control s)

undo (presses control z)

screenshot (takes a capture of the whole screen and saves it)


Ask other people in chat before you use:

quit (alt f4)

close tab (control w)

maximize (windows key arrow up)

minimize / show desktop (windowskey m)


Mouse:

click (left clicks mouse)

gclick (clicks more then frame works better for games)

grightclick (right clicks more then frame works better for games)

double click (left clicks 2 times fast)

hold mouse (holds left click for 6 sec)

hold mouse long (holds left click for 10 sec)

right click (rightclicks mouse)

middle click (clicks the middle mouse button/scroll wheel)

forward (presses the browser forward button)

back (presses the browser back button)

scroll up/down (scrolls up/down 60 pixels)


Mouse movement:

super left/right/up/down (moves pointer)

left/right/up/down (moves pointer)

light left/right/up/down (moves pointer less)

super light left/right/up/down (moves pointer even less)

go to {location} (moves pointer to a specific location example: go to 100 100) (note: resolution is 1920x1080)

go to center (moves mouse pointer to center of screen)


Dragging (good for paint)

drag up/down/left/right (moves mouse while holding left mouse button)

drag to {location} (drags pointer to a specific location example: drag to to 100 100)


where? (shows the position of the mouse)
